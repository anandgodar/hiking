## Data Sourcing Strategy

### Primary Sources (Free & Legal):

#### 1. **National Park Service (NPS.gov)**
- **All national parks** have official trail data
- GPS coordinates, elevation, distance
- Photos (public domain)
- Trail descriptions
- **Legal:** Public domain, can use freely

**Example:**
```
https://www.nps.gov/yose/planyourvisit/hiking.htm
```

#### 2. **US Forest Service (FS.USDA.gov)**
- National forest trails
- Wilderness areas
- GPS tracks
- **Legal:** Public domain

#### 3. **State Parks Websites**
- Each state has official trail data
- Usually public information
- Contact for permission if unsure

**Examples:**
- California: parks.ca.gov
- Colorado: cpw.state.co.us
- Washington: parks.wa.gov

#### 4. **USGS TopoView**
- Topographic maps
- Elevation data
- Geographic features
- **Legal:** Public domain

#### 5. **Local Hiking Clubs (With Permission)**
- Appalachian Mountain Club (AMC)
- Colorado Mountain Club (CMC)
- Mazamas (Oregon)
- Sierra Club chapters
- **Approach:** Email for partnership/data sharing

#### 6. **OpenStreetMap (OSM)**
- Crowd-sourced trail data
- GPS tracks
- **Legal:** Open Database License (attribute OSM)

**How to extract:**
```
https://www.openstreetmap.org/export
Select area ‚Üí Export GPX
```

#### 7. **AllTrails (Reference ONLY - Do NOT Copy)**
- ‚ùå DO NOT scrape or copy content
- ‚úÖ CAN use for trail names/locations to research
- ‚úÖ CAN use public NPS/USFS data they reference
- ‚ùå DO NOT copy descriptions, photos, or user content

### Secondary Sources:

#### 8. **Freelance Hikers**
- Hire local hikers to write original trail guides
- Cost: $50-100 per trail
- They provide: Photos, GPS track, firsthand description
- **Quality:** Best option for original content

#### 9. **User-Generated Content**
- Add review system (like you plan to)
- Users contribute photos, conditions, tips
- **Legal:** Have clear terms of service

#### 10. **Photography**
- **Unsplash** (currently using) - Free, high-quality
- **Pexels** - Free stock photos
- **Wikimedia Commons** - Public domain/Creative Commons
- **Purchase stock** - iStock, Shutterstock ($$)

## Content Production Workflow

### Option 1: AI-Assisted + Human Edit (Fastest)

**Step 1:** Gather NPS/USFS data
```bash
# Example: Yosemite trails
curl https://www.nps.gov/yose/planyourvisit/upload/traillist.pdf
# Extract trail names, distances, elevations
```

**Step 2:** Generate JSON with AI
```
Prompt: "Create JSON trail data for [Trail Name] in [State]:
- Research NPS.gov for official data
- Include lat/lon from Google Maps
- Add distance, elevation gain, difficulty
- Format as per template"
```

**Step 3:** Human review & edit
- Verify coordinates
- Check distances/elevations
- Add unique insights
- Test on site

**Cost:** ~$5-10 per trail (AI + human time)
**Speed:** 20-30 trails/day

---

### Option 2: Freelance Hikers (Highest Quality)

**Process:**
1. Post job on Upwork/Fiverr
2. "Need local hiker in [State] to document 10 trails"
3. Provide JSON template

5. Review and publish

**Cost:** $50-100 per trail
**Speed:** 2-5 trails/day per writer
**Quality:** Excellent (firsthand experience)

### Automation Scripts:

```javascript
// bulk-create-jsons.js
// Generate 100 JSON files from CSV of trail data

const fs = require('fs');
const csv = require('csv-parser');

const template = {
  name: "",
  slug: "",
  state: "Colorado",
  state_slug: "colorado",
  // ...
};

fs.createReadStream('colorado-trails.csv')
  .pipe(csv())
  .on('data', (row) => {
    const trail = {...template, ...row};
    fs.writeFileSync(
      `website/src/data/colorado/${trail.slug}.json`,
      JSON.stringify(trail, null, 2)
    );
  });
```
